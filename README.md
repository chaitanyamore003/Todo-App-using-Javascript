# To-Do App

A simple To-Do List web app built using **HTML**, **CSS**, and **JavaScript**.

## Features
- Add new tasks  
- Mark tasks as completed  
- Delete tasks  
- Saves tasks in browser using Local Storage  

## How to Use
1. Open `index.html` in your browser.  
2. Add your tasks in the input field.  
3. Click the checkbox to mark them as done or delete them when finished.  

## Technologies Used
- HTML  
- CSS  
- JavaScript
